# Add files to Tresorit
Opens https://send.tresorit.com

Here's what the script does:

## Features:

✅ Adds a green button labeled "📤" to Gmail webpages.

✅ Button is positioned in the bottom-left corner (fixed position)

✅ Clicking the button opens https://send.tresorit.com/ in a new popup window

✅ Includes hover effects for better UX

✅ High z-index (10000) to ensure it appears above other content

## To use this script:

Install the Tampermonkey extension for Chrome

Click the Tampermonkey icon → "Create a new script"

Copy the entire content from tresorit.user.js

Save the script
